## :book: Quick Start

!!! tip "Get up and running in 3 steps"

    === "Step 1: Install"

        ```bash
        pip install mkdocs-material
        ```

    === "Step 2: Create"

        ```bash
        mkdocs new my-project
        cd my-project
        ```

    === "Step 3: Serve"

        ```bash
        mkdocs serve
        ```

---

## :gear: Commands

| Command | Description |
|:--------|:------------|
| `mkdocs serve` | Start live-reloading dev server |
| `mkdocs build` | Build static site for production |
| `mkdocs gh-deploy` | Deploy to GitHub Pages |
| `mkdocs new [name]` | Create a new project |
